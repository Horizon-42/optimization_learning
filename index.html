Popular algorithms for unconstrained minimization

A visual, equation-first guide following the unconstrained minimization categories in the SciPy optimization tutorial. The project is built one algorithm at a time so each method can get a careful explanation and its own generated figures.

Roadmap

SciPy's unconstrained minimization section introduces derivative-free, quasi-Newton, Newton-CG, and trust-region Newton methods. This site uses the same category as its spine and adds visual intuition, practical rules of thumb, and clean mathematical notation for each algorithm.

Nelder-Mead Simplex

Derivative-free search that moves and reshapes a simplex using reflection, expansion, contraction, and shrink steps.

Finished in this step

BFGS

Quasi-Newton method that builds an inverse-Hessian approximation from gradient differences.

Next

Newton-CG

Uses gradients and Hessian-vector products to approximately solve the Newton system.

Trust-region Newton methods

Chooses steps inside a local region where the quadratic model is trusted.

Visualization Pipeline

Figures are generated with Python scripts in scripts/ and written to assets/. For example: 

conda run -n aviation env MPLCONFIGDIR=./.mplconfig python scripts/generate_nelder_mead_visuals.py
