Roadmap

The first completed block covered unconstrained multivariate local minimization. The guide now uses larger SciPy tutorial sections as the organizing layer, so new methods can be added without flattening every solver into one long list.

scipy.optimize.minimize

Local minimization of multivariate scalar functions

Local solvers refine one candidate point. The completed pages below cover the unconstrained algorithms from the SciPy tutorial plus the constrained methods that add bounds, equalities, and inequalities.

Nelder-Mead Simplex

Derivative-free search that moves and reshapes a simplex using reflection, expansion, contraction, and shrink steps.

Complete

BFGS

Quasi-Newton method that builds an inverse-Hessian approximation from gradient differences.

Complete

Newton-CG

Uses gradients and Hessian-vector products to approximately solve the Newton system.

Complete

trust-ncg

Trust-region Newton-CG method that constrains each quadratic-model step inside a radius.

Complete

trust-krylov

Solves the trust-region subproblem more accurately in a Krylov subspace.

Complete

trust-exact

Nearly exact trust-region method for medium-size problems with explicit Hessians.

Complete
